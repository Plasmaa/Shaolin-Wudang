## ğŸš€ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Plasmaa/Shaolin-Wudang.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Shaolin-Wudang
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
